# date-plugin v1.0.0-beta.1 发布说明

## 发布定位

这是 `date-plugin` 向 v1.0 正式版推进过程中的第一个发布候选骨架版本，目标是交付一套可集成、可测试、可继续演进的单日期选择器基础包。

## 本版本包含

- `DatePicker` 模块化公共 API
- legacy 手动初始化与 auto 初始化入口
- 日期格式化、本地化、约束与键盘导航
- Today / Clear 操作区
- 桌面端浮层定位与移动端 sheet 模式
- overlay、锁滚、safe-area 与 reduced-motion 支持
- `setMonth(year, month)` 与 `onMonthChange`
- 浏览器测试页与回归页

## 已知限制

- 当前仍以浏览器回归页作为主要验证方式
- 当前已具备本机 preflight 与浏览器自动回归，但尚未提供编译后的分发构建目录
- 当前仅接入最小浏览器 CI，尚未提供自动发布和完整构建流水线

## 下一步建议

- 继续补齐版本发布脚本与发布自动化
- 增加许可证与仓库元数据之外的 npm 发布细节
- 在正式版前完成最终兼容性检查与文档清理
