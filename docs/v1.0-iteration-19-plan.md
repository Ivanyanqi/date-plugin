# date-plugin v1.0 第十九轮计划

## 本轮目标

启动首页产品化升级，把当前 [index.html](../index.html) 从简单 demo 提升为 interactive showcase 首页，为后续官网化和文档化演进打基础。

## 范围

- 新增首页设计说明文档
- 新增首页实现计划文档
- 重写 [index.html](../index.html)
- 增加首页专属展示样式与脚本
- 保持现有回归链稳定

## 本轮不做

- 不修改发布日期流程
- 不新增新的核心日期功能
- 不重写 [regression.html](../regression.html)

## 验证方式

1. 本地打开 [index.html](../index.html)
2. 验证试玩区配置切换
3. 回归 `npm run test:ci`

## 预期产出

- 一个更像产品首页而不是测试页的 interactive showcase
- 开发者和产品/设计评审都能立即理解组件价值
