# date-plugin v1.0 第七轮计划

## 本轮目标

把移动端 sheet 模式从“布局成立”推进到“交互成立”，补齐遮罩层、页面锁滚和移动端关闭路径，让小屏体验不再只是一个固定到底部的面板。

## 范围

### 1. 移动端遮罩层

- sheet 模式打开时显示半透明 backdrop
- backdrop 只在移动端模式下出现
- backdrop 点击后关闭当前面板

### 2. 页面锁滚

- sheet 模式打开时锁定 `body` 滚动
- sheet 关闭、销毁或切回桌面模式后恢复滚动
- 避免重复 lock / unlock 导致 body 样式紊乱

### 3. 生命周期兼容

- overlay close 与现有 `close()`、outside close、destroy 行为兼容
- 不影响桌面模式的 `top / bottom` 浮层逻辑
- 不影响已有输入同步、Today/Clear、键盘交互和回调

## 本轮不做

- 不做关闭动画
- 不做拖拽关闭
- 不做焦点陷阱
- 不做页面 inert 管理
- 不做平台级安全区适配

## 风险点

- 锁滚需要在多实例和销毁路径下保持成对释放
- overlay 点击关闭不能误伤桌面模式和普通文档点击关闭逻辑
- resize 从移动切回桌面时必须正确移除 backdrop 和解锁 body

## 验证方式

1. 新增 mobile overlay / scroll lock 测试页
2. 回归 mobile sheet / placement 测试
3. 回归输入同步、键盘导航、ARIA、公共 API
4. 回归 [regression.html](../regression.html)

## 预期产出

- 移动端 sheet 具备更完整的打开和关闭体验
- 后续可以继续进入动画、手势和安全区适配
- v1.0 的移动端基础交互已经基本闭环
