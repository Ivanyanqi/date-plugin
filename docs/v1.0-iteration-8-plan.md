# date-plugin v1.0 第八轮计划

## 本轮目标

把移动端 sheet 的展示和关闭过程做成完整可感知的交互层，补齐开关动画状态和安全区底部留白，让移动端体验更接近真实产品。

## 范围

### 1. 开关动画状态

- sheet 打开时暴露 `opening -> open`
- sheet 关闭时暴露 `closing -> closed`
- overlay 与面板保持一致的状态节奏

### 2. 安全区留白

- 底部操作区为安全区预留额外 padding
- 使用 `env(safe-area-inset-bottom, 0px)` 作为默认底部补偿来源
- 保证无安全区环境下仍有合理的 footer 底部间距

### 3. 生命周期稳定性

- 动画状态与 overlay / 锁滚兼容
- 销毁实例时清理动画定时器和 frame 句柄
- 不影响桌面模式和既有回归用例

## 本轮不做

- 不做拖拽手势动画
- 不做弹簧物理效果
- 不做 reduced-motion 偏好适配
- 不做安全区左右刘海特殊适配
- 不做页面级 focus trap

## 风险点

- 动画定时器和 open frame 不能在重复开关时残留
- overlay closing 期间必须继续维持锁滚，直到动画结束
- safe-area 留白不能破坏桌面 footer 布局

## 验证方式

1. 新增 mobile animation / safe-area 测试页
2. 回归 mobile overlay / sheet 测试
3. 回归输入同步、键盘导航、ARIA、公共 API
4. 回归 [regression.html](../regression.html)

## 预期产出

- 移动端 sheet 的进入和退出更自然
- 后续可以继续进入 reduced-motion、手势和安全区细化
- v1.0 的移动端视觉与交互收口更完整
